# Payment Service

Интеграция с платежными шлюзами (Stripe, PayPal и т.д.), лог транзакций, управление выплатами продавцам.

## Начало работы

1. Установить зависимости: `npm install`
2. Создать и заполнить `.env` (пример в `.env.example`)
3. Запустить сервис в режиме разработки: `npm run start:dev`

## Эндпоинты

- GET `/api/payments` — список транзакций
- GET `/api/payments/:id` — детальная информация о транзакции
- POST `/api/payments` — создать транзакцию (платеж)
- PUT `/api/payments/:id` — обновить статус транзакции
- DELETE `/api/payments/:id` — удалить запись о транзакции
