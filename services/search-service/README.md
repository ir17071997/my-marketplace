# Search Service

Поиск и фильтрация по товарам и магазинам, автодополнение, аналитика поисковых запросов.

## Начало работы

1. Установить зависимости: `npm install`
2. Создать и заполнить `.env` (пример в `.env.example`)
3. Запустить сервис в режиме разработки: `npm run start:dev`

## Эндпоинты

- GET `/api/search?q={keyword}` — поиск товаров и магазинов
- GET `/api/search/suggestions?q={keyword}` — автодополнение
