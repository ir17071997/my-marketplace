# Order Service

Управление заказами: создание заказа, отслеживание статуса, управление заказами (покупатель/продавец/админ), возвраты и отмены.

## Начало работы

1. Установить зависимости: `npm install`
2. Создать и заполнить `.env` (пример в `.env.example`)
3. Запустить сервис в режиме разработки: `npm run start:dev`

## Эндпоинты

- GET `/api/orders` — список всех заказов
- GET `/api/orders/:id` — получить заказ по ID
- POST `/api/orders` — создать заказ
- PUT `/api/orders/:id` — обновить заказ
- DELETE `/api/orders/:id` — удалить заказ
