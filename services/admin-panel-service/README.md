# Admin Panel Service

Административная панель: модерация пользователей, товаров и магазинов, управление платформенными настройками, система тикетов поддержки и маркетинговые баннеры.

## Начало работы

1. Установить зависимости: `npm install`
2. Создать и заполнить `.env` (пример в `.env.example`)
3. Запустить сервис в режиме разработки: `npm run start:dev`

## Эндпоинты

- GET `/api/admin/logs` — история административных действий
- POST `/api/admin/logs` — зарегистрировать действие (action, payload)
- GET `/api/admin/config` — получить платформенные настройки
- POST `/api/admin/config` — обновить настройки
- GET `/api/admin/tickets` — список тикетов поддержки
- POST `/api/admin/tickets` — создать тикет поддержки
