# Analytics Service

Сбор и отображение статистики: продажи, посещения, тренды, отчеты для продавцов и администраторов.

## Начало работы

1. Установить зависимости: `npm install`
2. Создать и заполнить `.env` (пример в `.env.example`)
3. Запустить сервис в режиме разработки: `npm run start:dev`

## Эндпоинты

- GET `/api/analytics/sales?period=2025-07` — отчет о продажах за период
- GET `/api/analytics/visitors?period=2025-07` — отчет о посетителях
- GET `/api/analytics/trends?period=2025-07` — аналитика трендов
